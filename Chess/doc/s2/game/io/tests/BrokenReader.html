BrokenReader

class BrokenReader(realReader: Reader, throwExceptionAt: Int) extends Reader

This class is used to artificially create problems. :)

The class will take a normal Reader object and serve all data read from that Reader until a given breaking point is reached. At this point the read method will throw an IOException.

The constructor constructs a new BrokenReader from a given normal Reader.

Value parameters:
realReader

The real reader object used to serve data until the breaking point is reached.

throwExceptionAt

The breaking point position.
